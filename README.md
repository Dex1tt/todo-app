📝 To-Do App

Простое приложение для управления задачами на чистом HTML, CSS и JavaScript.

Возможности


✅ Добавление задач (кнопкой или клавишей Enter)
✓ Отметка задачи выполненной
🗑 Удаление задачи
🔍 Фильтры: все / активные / выполненные
📊 Счётчик выполненных задач
💾 Сохранение в localStorage (задачи остаются после обновления страницы)
🕳 Пустое состояние если задач нет


Структура файлов

├── index.html
├── style.css
└── script.js

Как запустить

Просто открой index.html в браузере — никаких зависимостей и сборки не нужно.

Что использовалось


HTML — разметка
CSS — стили и адаптация
JavaScript — логика приложения
localStorage — хранение задач в браузере
