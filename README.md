Focus


Все задачи на своём месте



Минималистичный таск-менеджер на vanilla JS с локальным хранилищем.

Возможности


✅ Добавление задач кнопкой или клавишей Enter
✓ Отметка задачи выполненной
🗑 Удаление задачи
🧹 Очистка всех выполненных одним кликом
🔍 Фильтры: все / активные / выполненные
📊 Счётчик выполненных задач
💾 Сохранение в localStorage
🕳 Пустое состояние если задач нет


Структура файлов

├── index.html
├── style.css
└── script.js

Как запустить

Открой index.html в браузере — никаких зависимостей и сборки не нужно.
